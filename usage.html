使い方

このページでは基本的な使い方を紹介します。 詳細はよくある質問を参照してください。

メーリングリストの作成

enkai@quickml.com のような好きなアドレスにメールを送れば、 新しいメーリングリストを作れます。 enkai@gotanda.quickml.com のように名前にサブドメインが 入ったメーリングリストを作ることもできます。 

Subject: 宴会メーリングリスト作りました
To: enkai@quickml.com                ← 作りたいMLのアドレス
From: satoru@example.com             ← 自分のアドレス
Cc: masui@example.com                ← 参加者リスト

突然ですが、宴会好きのメーリング     ← 本文
リストを作ってみました。

From: と Cc: のアドレスがメーリングリストに登録され、 登録されたメンバーにメールが送られます。

メーリングリストへ投稿

作成したメーリングリストのアドレスにメールを送れば投稿できま す。メンバー以外からのメールは受け付けません。 

Subject: 今晩どう?                     
To: enkai@quickml.com                ← MLのアドレス
From: satoru@example.com             ← 自分のアドレス

さっそくですが今晩はどうでしょうか。 ← 本文

新しいメンバーの追加

すでに参加しているメンバーは 新しいメンバーを Cc: で追加することができます。 

Subject: 小松君を追加
To: enkai@quickml.com                  ← MLのアドレス
From: satoru@example.com               ← 自分のアドレス
Cc: komatsu@example.com                ← 新メンバーのアドレス

小松君を新しいメンバーとして           ← 本文
呼んできました。

メーリングリストへ参加

すでに参加しているメンバーを知っていれば、 そのメンバーを Cc: で指定すれば メーリングリストに参加できます。 参加のために送ったメールはメーリングリストに流れるので、 注意しましょう。 

Subject: 参加します
To: enkai@quickml.com                ← MLのアドレス
From: tsuka@example.com              ← 自分のアドレス
Cc: masui@example.com                ← 参加メンバーのアドレス

僕を忘れてもらっては困ります。       ← 本文

メーリングリストから退会

メーリングリストに空メールを送ると、メーリ ングリストから退会できます (空メールはMLには流れません)。 シグネチャをつけないように注意しましょう。 

Subject: 退会します
To: enkai@quickml.com                ← MLのアドレス
From: satoru@example.com             ← 自分のアドレス

                                     ← 空の本文

空メールを送れない場合は本文に「退会」とだけ書いたメールを MLに送ってください (MLには流れません)。

登録アドレスの変更

メーリングリストに登録されているアドレスを変更したいときは、 新しいアドレスを追加してから 古いアドレスで退会して下さい。

メンバーの削除

届かないアドレスを登録してしまったなどの理由でメンバーのアド レスを削除したい場合は Cc: に削除したいアドレスを指定して メーリングリストに空メールを送ると、指定し たアドレスをメーリングリストから削除できます。 シグネチャをつけないように注意しましょう。 

Subject: 削除します
To: enkai@quickml.com                ← MLのアドレス
From: satoru@example.com             ← 自分のアドレス
Cc: fugo-masui@pitecan..com          ← 削除するアドレス

                                     ← 空の本文

嫌いなメンバーを勝手に削除するといった悪用はやめましょう。

メーリングリストへ復帰

退会したメーリングリストに再びメールを送ると、メーリ ングリストに復帰できます。 

Subject: 復帰しました
To: enkai@quickml.com                ← MLのアドレス
From: satoru@example.com             ← 自分のアドレス
